# Rafi Fadhila Putra - Digital Marketing Professional Portfolio

A modern, responsive portfolio website showcasing professional digital marketing experience, tools expertise, brand collaborations, and campaign management capabilities.

## 🎯 Features

### Interactive Components
- **About Me Slider**: Navigate through professional profiles
- **Tools & Platforms**: Comprehensive breakdown of software proficiency across 7 categories
- **Brands Experience**: Horizontal carousel showcasing brand collaborations
- **Campaign Projects**: Auto-scrolling showcase of campaign types with 3D perspective effects
- **Modal System**: Click-to-view brand campaign details
- **Responsive Design**: Mobile-optimized layout

### Design Elements
- Dark theme with orange accent color (#ff5500)
- Smooth scrolling and transitions
- Backdrop blur effects on header
- Gradient text effects on logo
- Progress bars and indicators
- Hover animations and micro-interactions

## 📁 Project Structure

## 🚀 Deployment

This portfolio is deployed on Vercel for seamless CI/CD:

1. **Push to GitHub**: Changes are automatically detected
2. **Vercel Build**: Triggers instant deployment
3. **Live Update**: Changes go live immediately

### Deployment URL
- [rafi-fadhila-putra-personal-blog.vercel.app](https://rafi-fadhila-putra-personal-blog.vercel.app)
